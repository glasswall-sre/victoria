victoria.config.load_plugin_config(plugin: Plugin, cfg: Config) → object¶

Load the config of a plugin from the main Victoria config.

Parameters

  • plugin (Plugin) – The plugin to load the config for.

  • cfg (Config) – The config file to load the config from.

Returns

The loaded config object. It will be the same type as whatever

the plugin’s config marshmallow schema will marshal it to, or None if there was some error loading the plugin config.

Return type

object

Raises

ValueError – if the plugin didn’t have a config schema.
