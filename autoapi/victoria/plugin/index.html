victoria.plugin.load(plugin_name: str) → Plugin¶

Load a plugin by name.

Plugins are installed python modules with a name starting with ‘victoria_’. For example: ‘victoria_config’. Plugin packages should define a plugin object in their __init__.py and use it to wire up plugin functionality.

A plugin can optionally define a marshmallow schema that can be used for validating its config. Plugin configs will be located in the ‘plugins_config’ object of the Victoria config, under an object with the same name as the plugin.

Parameters

plugin_name (str) – The name of the plugin package to load. Plugin packages must be in the format “victoria_{plugin_name}”.

Returns

the plugin object as loaded from the module, or None if there was

some error whilst loading.

Return type
